# DXスキル進捗ダッシュボード

PCやスマホから同じデータにアクセスできる、Googleスプレッドシート連携型の学習進捗管理ダッシュボードです。

## 📋 ファイル構成

| ファイル名 | 説明 |
|-----------|------|
| `dx_skill_dashboard.html` | 進捗管理ダッシュボード（メインUI） |
| `google_apps_script.gs` | Google スプレッドシート連携用スクリプト |
| `README.md` | このドキュメント |

## ✨ 主な機能

- 📊 **学習テーマ別の進捗管理** - タブで分類された学習記録
- 📅 **日付管理** - いつ何を学習したかを記録
- ⏱️ **学習時間の集計** - テーマごとの総学習時間を自動計算
- 🎥 **参考動画リンク** - テーマごとにYouTube動画を登録
- ☁️ **クラウド同期** - Googleスプレッドシートでデータを管理
- 📱 **マルチデバイス対応** - PC・スマホから同じデータにアクセス
- 💾 **エクスポート機能** - JSONファイルでバックアップ可能

## 🚀 セットアップ手順

### Step 1: Googleスプレッドシートの作成

1. [Google Drive](https://drive.google.com) を開く
2. 「新規」→「Google スプレッドシート」→「空白のスプレッドシート」
3. スプレッドシート名を「**DXスキルアップ記録簿**」に変更
4. URLからスプレッドシートIDをコピー

```
URL例: https://docs.google.com/spreadsheets/d/1ABC123xyz.../edit
                                             ↑↑↑↑↑↑↑↑↑↑↑
                                             この部分がID
```

### Step 2: Google Apps Scriptの設定

1. スプレッドシートで「**拡張機能**」→「**Apps Script**」を開く
2. デフォルトのコードをすべて削除
3. `google_apps_script.gs` の内容をコピー＆ペースト
4. **SPREADSHEET_ID** を実際のIDに置き換え:

```javascript
const SPREADSHEET_ID = 'ここにコピーしたIDを貼り付け';
```

5. 💾「保存」アイコンをクリック

### Step 3: 初期化の実行

1. 関数セレクタで「**testInit**」を選択
2. ▶「**実行**」ボタンをクリック
3. 初回実行時は認証を求められます:
   - 「権限を確認」をクリック
   - Googleアカウントを選択
   - 「詳細」→「〇〇（安全ではないページ）に移動」
   - 「許可」をクリック
4. スプレッドシートに「DXスキルアップデータ」「更新ログ」シートが作成されます

### Step 4: ウェブアプリとしてデプロイ

1. 「**デプロイ**」→「**新しいデプロイ**」をクリック
2. ⚙️ 歯車アイコン →「**ウェブアプリ**」を選択
3. 以下のように設定:

| 設定項目 | 値 |
|---------|-----|
| 説明 | DXスキルダッシュボードAPI |
| 次のユーザーとして実行 | **自分** |
| アクセスできるユーザー | **全員** |

4. 「**デプロイ**」をクリック
5. 表示される**URL**をコピー

```
例: https://script.google.com/macros/s/AKfycbw.../exec
```

### Step 5: HTMLファイルにAPI URLを設定

`dx_skill_dashboard.html` をテキストエディタで開き、以下の部分を編集:

```javascript
// ============================================
// 設定 - Google Apps ScriptのウェブアプリURLを設定
// ============================================
const API_URL = 'https://script.google.com/macros/s/AKfycbw.../exec';
```

## 📊 スプレッドシートの構成

### メインシート: 「DXスキルアップデータ」

| 列 | 内容 |
|----|------|
| A | データ（JSON形式） |
| B | 最終更新日時 |
| C | タブ数 |
| D | 総記録数 |
| E | 総学習時間 |

### ログシート: 「更新ログ」

| 列 | 内容 |
|----|------|
| A | タイムスタンプ |
| B | 操作（保存/読込） |
| C | ユーザー |
| D | 詳細 |

## 🎯 使い方

### 学習記録の追加

1. 対象のテーマタブを選択
2. フォームに以下を入力:
   - 日付
   - テーマ（学習内容）
   - 説明（任意）
   - 勉強時間
   - 媒体（Udemy Business / YouTube / その他）
   - URL（任意）
3. 「＋ 記録を追加」をクリック
4. 自動的にスプレッドシートに保存されます

### 学習記録の編集・削除

- **編集**: 記録の「✏️ 編集」ボタンをクリック
- **削除**: 記録の「🗑️ 削除」ボタンをクリック

### タブの操作

- **追加**: 「＋ タブを追加」ボタン
- **編集**: タブをダブルクリック
- **削除**: タブの×ボタン、または編集画面から

### 参考動画の管理

各タブの下部にある「🎥 おすすめ学習動画」セクションで:
- 「＋ 動画追加」で新しい動画を登録
- 動画カードにホバーして✏️/🗑️で編集・削除

## ⚠️ 注意事項

### セキュリティについて

- ウェブアプリのアクセス権限を「全員」に設定する必要があります（CORS制限のため）
- ただし、URLを知らない人はアクセスできません
- 機密性の高いデータは保存しないでください

### データの保存について

- すべての操作は自動的にスプレッドシートに保存されます
- ローカル（ブラウザ）にもバックアップが保存されます
- 定期的に「💾 エクスポート」でJSONバックアップを推奨

### ブラウザ互換性

- Chrome、Firefox、Edge、Safari の最新版で動作確認済み

## 🔧 トラブルシューティング

### 「接続エラー」が表示される

1. API URLが正しいか確認
2. Apps Scriptがデプロイされているか確認
3. デプロイの「アクセスできるユーザー」が「全員」になっているか確認

### データが読み込まれない

1. スプレッドシートに「DXスキルアップデータ」シートがあるか確認
2. Apps Scriptで`testLoad()`を実行してエラーがないか確認

### 保存されない

1. ブラウザのコンソール（F12）でエラーを確認
2. Apps Scriptの実行ログを確認
